## 可用工具

| 工具 | 功能 | 参数 |
|------|------|------|
| `listDomesticProducts` | 获取所有国内产品列表 | channel (可选): internet/offline/all |
| `searchDomesticProducts` | 多条件搜索产品 | keyword, productType, channel, company, hotLabel |
| `getDomesticProductDetail` | 获取产品详情 | productCode (必填) |
| `getDomesticProductCategories` | 获取分类体系 | 无 |

## 使用场景

### 场景一：查询互联网渠道产品
当用户说"查询互联网保险产品"或"线上可投保的产品"时：
```
调用 listDomesticProducts(channel="internet")
```

### 场景二：搜索特定公司产品
当用户说"平安健康有什么产品"或"中英人寿的产品"时：
```
调用 searchDomesticProducts(company="平安健康")
```

### 场景三：搜索特定类型产品
当用户说"国内重疾险"或"年金险有哪些"时：
```
调用 searchDomesticProducts(productType="P04")  // 重疾险
调用 searchDomesticProducts(productType="P03")  // 年金险
```

### 场景四：获取产品详情
当用户提供产品编号或需要了解某产品详细信息时：
```
调用 getDomesticProductDetail(productCode="179403")
```

### 场景五：查看分类体系
当用户想了解产品分类、有哪些公司、有哪些标签时：
```
调用 getDomesticProductCategories()
```


## 输出规范

1. **禁止直接输出 JSON** — 必须解析、格式化为表格
2. **优先使用表格** 呈现可对比数据（产品名称、公司、类型、简介）
3. **信息缺失标注** "数据暂未披露"
4. **渠道说明** 
   - 互联网渠道 (productChannel=1): 支持在线投保，流程便捷
   - 线下渠道 (productChannel=2): 需顾问协助投保，服务更全面
5. **产品状态** productStatus=3 表示在售状态
